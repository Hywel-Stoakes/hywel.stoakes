# Introduction

This is the repository for my personal website www.hywel-stoakes.net and holds all of the text and data for a Hugo-powered website (https://gohugo.io). It uses the academic theme (https://themes.gohugo.io/academic/) which is very flexible in terms of adding data and documents. I am not the world's most diligent blogger but come back for updates on my research regarding speech and endangered languages.
